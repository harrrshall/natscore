## Primary training data

- **SpeechJudge-Data** — `huggingface.co/datasets/RMSnow/SpeechJudge-Data`
  - 99K human preference pairs over modern neural TTS output (Nov 2025 release)
  - License: **CC-BY-NC-4.0**
  - Train split: ~42K pairs
  - High-consensus subset (rater agreement >40%): ~31K pairs
  - Languages: en, zh, code-switching
  - TTS models covered: CosyVoice2, F5-TTS, MaskGCT, Llasa, Inits, others
  - Subsets: `regular` + `expressive`

**Gated access** — run `huggingface-cli login` and accept the dataset terms
on the HF page before the loader will work.
